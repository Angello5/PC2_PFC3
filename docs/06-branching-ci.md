# Branching, Commits y CI/CD

## Ramas

- `main`: version estable desplegable.
- `develop`: integracion del sprint.
- `feature/base-datos`: SQL y seed.
- `feature/autenticacion`: Supabase Auth y perfiles.
- `feature/dashboard`: panel y estadisticas.
- `feature/despliegue`: Vercel, Railway y CI/CD.

## Commits en español

```txt
feat: agregar esquema de base de datos y datos iniciales
feat: agregar flujo de autenticacion con supabase
feat: agregar registro de trabajadores
feat: agregar flujo de qr para donaciones
feat: agregar reporte de incidentes
feat: agregar panel de estadisticas para organizacion
ci: agregar pipeline de github actions
chore: agregar configuracion de despliegue
docs: agregar evidencias del informe final
```
